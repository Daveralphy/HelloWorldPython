# Overview

As a software engineer who builds products end to end, from Figma wireframes to deployed apps in React, Next.js, Flutter, and Python, I'm always looking for ways to sharpen my fundamentals across different languages and stacks. This project is part of that ongoing practice: making sure my development environment, version control workflow, and documentation habits are solid before diving into more complex builds.

This software is a simple Python program that prints a personalized greeting to the console. While the program itself is intentionally minimal, it represents the first checkpoint in a broader learning process, confirming that my local setup, Git workflow, and GitHub publishing process all work the way they should before I build anything more substantial on top of them.

My purpose in creating this was to validate my toolchain end to end: writing code locally in VS Code, tracking it with Git, and publishing it to a public GitHub repository, the same workflow I'll rely on for every project going forward, just like I do with the larger products I ship at WindVeal.

[Software Demo Video](https://youtu.be/MKYEbn5goho)

# Development Environment

I developed this software using Visual Studio Code as my editor, with Git for version control and GitHub for hosting the public repository. The code was written and executed using Python 3, run directly from the integrated terminal.
